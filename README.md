## Local preview
Just open `index.html` in a browser, or run a tiny server:
```
python3 -m http.server 8000
```
Then visit http://localhost:8000.

## Deploy to GitHub Pages
1. Create a new GitHub repo (e.g. `portfolio`).
2. Drag the contents of this folder into the repo (or `git push` them).
3. Repo → **Settings** → **Pages** → set Source = `Deploy from a branch`, Branch = `main` (root) → **Save**.
4. Wait ~1 minute. Your site will be live at `https://<your-username>.github.io/portfolio/`.

## Editing content
All copy lives in **`shared/content.js`** — name, role, artifacts, KSA, reflections, etc. Edit there, refresh, done. No build step.
